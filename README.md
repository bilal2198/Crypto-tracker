# Crypto Tracker Dashboard

A sleek, real-time cryptocurrency price tracker with automated trading signals. Built with vanilla JavaScript and featuring a cyberpunk-inspired interface, it monitors Bitcoin, Ethereum, and Solana prices with live market updates every 30 seconds.

## Features

✨ **Real-Time Price Tracking**
- Live cryptocurrency prices for Bitcoin (BTC), Ethereum (ETH), and Solana (SOL)
- 24-hour price change indicators with visual positive/negative styling
- Data refreshed every 30 seconds from CoinGecko API

🤖 **Automated Trading Signals**
- Smart signal generation based on 24-hour price movements
- BUY signal: Price increase > 2%
- SELL signal: Price decrease > -2%
- HOLD signal: Price movement between -2% and +2%

📊 **Activity Log**
- Real-time bot activity feed showing market scans and updates
- Timestamp for each log entry
- Last 6 updates displayed

🎨 **Cyberpunk UI**
- Dark theme with neon green accents
- Responsive card-based layout
- Live status indicator with pulse animation
- Professional monospace font styling

## Technical Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with animations and gradients
- **JavaScript (ES6+)** - Async/await, Fetch API
- **CoinGecko API** - Free cryptocurrency data

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection (for API calls)

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. The dashboard will automatically start fetching and displaying live market data

```bash
# No installation required - just open the HTML file
open index.html
```

## How It Works

1. **Initial Load**: Dashboard displays placeholder data while initializing
2. **Market Scan**: Every 30 seconds, the app fetches current prices from CoinGecko
3. **Price Update**: Prices and 24-hour changes are updated in real-time
4. **Signal Generation**: Trading signals are calculated based on price movements
5. **Activity Logging**: Each update is logged with timestamp in the activity feed

## API Reference

The app uses the free CoinGecko API:
```
https://api.coingecko.com/api/v3/simple/price?ids=bitcoin,ethereum,solana&vs_currencies=usd&include_24hr_change=true
```

**Rate Limit**: Free tier allows requests without authentication. No API key required!

## Customization

### Adding More Cryptocurrencies

To add more coins to the tracker:

1. Add a new card section in `index.html`:
```html
<div class="card">
    <div class="card-header">
        <span class="coin-name">COIN NAME</span>
        <span class="coin-badge">SYMBOL</span>
    </div>
    <div class="price" id="coin-price">---</div>
    <div class="change" id="coin-change">...</div>
    <div class="signal-box">
        SIGNAL: <span class="signal" id="coin-signal">READING...</span>
    </div>
</div>
```

2. Update the `coins` array in `script.js` with the new cryptocurrency ID from CoinGecko

### Changing Update Frequency

Edit the interval in `script.js` (currently 30000 milliseconds):
```javascript
setInterval(getPrices, 30000); // Change to desired milliseconds
```

### Signal Thresholds

Modify the `getSignal()` function in `script.js` to adjust buy/sell sensitivity:
```javascript
function getSignal(change) {
    if (change > 5) return { text: 'BUY', cls: 'buy' };      // Changed from 2
    if (change < -5) return { text: 'SELL', cls: 'sell' };   // Changed from -2
    return { text: 'HOLD', cls: 'hold' };
}
```
